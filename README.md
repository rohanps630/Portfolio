# Rohan P. Suresh - Portfolio

This is the personal portfolio website of Rohan P. Suresh, a Senior Full Stack Developer specializing in React.js, React Native, Node.js, and AI-driven development.

## Project Info

This project is a modern, responsive web application built to showcase my skills, experience, and projects.

## Tech Stack

This project is built with:

- **Vite** - Next Generation Frontend Tooling
- **TypeScript** - Typed JavaScript at Any Scale
- **React** - The Library for Web and Native User Interfaces
- **Tailwind CSS** - Rapidly build modern websites without ever leaving your HTML
- **shadcn/ui** - Beautifully designed components built with Radix UI and Tailwind CSS

## Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository:**

    ```bash
    git clone <YOUR_GIT_URL>
    cd <YOUR_PROJECT_NAME>
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Start the development server:**

    ```bash
    npm run dev
    ```

4.  **Build for production:**

    ```bash
    npm run build
    ```
